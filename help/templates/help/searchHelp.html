Quick Search
EHFPI support four types to search:
  • Gene/Gene Product
    • A gene/protein can be queried by its Gene Symbol, previous name, Synonyms, Entrez gene id, UniprotKB ID and Ensembl gene ID.
  • Pathogen
    • A pathogen can be queried by its full name, abbreviation, pathogen aliases, and taxonomy information (strain, species, genus, family, group and kingdom).
  • Publication
    • A publication can be queried by its PUBMED id, title, journal, author, publication year.
  • EHFPI Acc
    • EHFPI Acc field is the identifier of EHFPI entry.
Quick search support autocomplete, if you choose a search type from the select, the autocomplete only display corresponding information. All field related to the query will be displayed by default. Note for each search type only the first five results will be displayed.
Advanced Search
You can combine the following five criteria to perform advanced search.
  • Gene/Gene Product
    • You are recommended to provide a list of gene symbol, entrez gene id, uniprotkb id, previous name and synonyms, separate by comma, e.g. cab5,cbx4. Or you can input one gene per line.
    Mixed id list is also supported.
  • Organism
    • Choose the kingdom and then select the corresponding species. You can select multiple entries.
  • Publication
    • You can filter author, journal and year field.
  • Screen Scope
    • Screen scope of the RNAi screen.
  • Phenotype
    • Phenotype of the essential host factor.
Results that match all of the set conditions will be shown by default. You can change the logic to 'any' at page bottom (left). Any of the existing conditions can be removed using the minus image button.
Customize Search Results
If you want to change the default displayed results according to your own research interest, please click the 'Custom Display Columns' button above the Result Table (right).
Then, choose the columns you want to display. Note: ehfpi acc, gene symbol, entrez gene id, pathogen (strain) and publication title are default displayed columns.
Change the order of a specific columns by clicking the table title, e.g. EHFPI ACC. The first click will result in ascending order, and second click descending order.
Data Export of Search Results
You can select all records (including all EHFPI information items) or the interested records of on the current page, then click the 'Download' button to obtain the results.
You can also choose the columns you want to for download. The file format is csv.
Analyze Search Results
Conveniently, you can perform analysis for all or selected EHF records by click the button above the search result. More details can be found in analysis part.
